# claw

**OpenClaw** — the open-source personal AI assistant. Multi-channel, multi-model, extensible. Run it on your own devices.

> This is the SuperInstance fork of [OpenClaw](https://github.com/openclaw/openclaw), used as the cellular logic engine in the fleet infrastructure.

## What This Is

This fork tracks upstream OpenClaw with SuperInstance-specific configurations and extensions. For the main project, see [openclaw/openclaw](https://github.com/openclaw/openclaw).
