## Getting Started

### New to Claw?

Welcome! Here's how to get started:

1. **Explore the Project:**
   - Read the [README](../README.md)
   - Check out the [documentation](https://claw.superinstance.ai)
   - Browse [example agents](../examples/)

2. **Join the Community:**
   - Star the repository on GitHub
   - Join our [Discussions](https://github.com/SuperInstance/claw/discussions)
   - Subscribe to updates

3. **Set Up Your Environment:**
   - Follow the [installation guide](../README.md#installation)
   - Run the examples
   - Build your first agent

### First Steps

**For Developers:**

**Documentation:**

- Add examples
- Fix typos and errors

**Tests:**
- Increase test coverage

## Community Events

### Regular Events

**Office Hours** (Monthly, First Thursday)
- Live Q&A with maintainers
- Project updates
- Roadmap discussion
- 14:00 UTC / 9:00 AM EST

**Contributing Sprints** (Monthly, Third Saturday)
- Focused contribution time
- Mentorship available
- Good for beginners
- 12:00 UTC / 7:00 AM EST

**Release Parties** (Quarterly)
- Celebrate new releases
- Feature highlights
- Contributor recognition
- After major releases

**Annual Summit** (Yearly)
- Year in review
- Future roadmap
- Guest speakers
- Community awards

### How to Participate

**Join Events:**
- Watch [Announcements](https://github.com/SuperInstance/claw/announcements)
- Add calendar events
- Join video calls (link in event)

**During Events:**
- Ask questions
- Share your work
- Provide feedback
- Meet other contributors

**After Events:**
- Follow up on discussions
- Work on action items
- Share takeaways

Each month, we highlight contributors who have made significant impact:

**Featured Contributors:**

**How to Get Featured:**

- Create high-impact work

**Top Contributors:**
- Listed in [CONTRIBUTORS.md](CONTRIBUTORS.md)

**Criteria:**
- 10+ merged PRs
- 50+ helpful issues/PRs reviewed

## Getting Help

### Troubleshooting

**Before Asking:**
1. Check [TROUBLESHOOTING.md](../TROUBLESHOOTING.md)
2. Search existing [Issues](https://github.com/SuperInstance/claw/issues)
3. Search [Discussions](https://github.com/SuperInstance/claw/discussions)

**When Asking:**
- Provide context and error messages
- Share minimal reproduction code
- Include version information
- Be patient and respectful

**Where to Ask:**
- **Quick Questions:** GitHub Discussions
- **Bug Reports:** GitHub Issues
- **Security Issues:** security@superinstance.ai
- **Code of Conduct:** conduct@superinstance.ai

### Mentorship

**New Contributors:**
- Check [good first issues](https://github.com/SuperInstance/claw/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)
- Join contributing sprints
- Ask for help in Discussions
- Request a mentor

**Experienced Contributors:**
- Become a mentor
- Review pull requests
- Answer questions
- Write documentation
